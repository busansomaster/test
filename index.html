Install BUSANSO NATION

Install the BUSANSO NATION app on your Android phone, tablet, or Smart TV for faster streaming and fullscreen viewing.

If install button does not appear:

  1. Open website in Google Chrome
  2. Tap the 3 dots menu
  3. Select Install App or Add to Home Screen
  4. Confirm installation
Compatible with Android phones, tablets, Android TV, Chromecast and Smart TVs.
